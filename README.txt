RVU Admissions Chatbot

This project simulates an intelligent chatbot for RV University admissions. Originally built using Botpress, this version is a Python terminal app for academic presentation purposes.

Features

- Chat interface for prospective students
- Menu options: Courses, Fees, Code of Conduct, Contact, AI Chat
- Simulated AI interaction
- Clean modular Python code

How to Run

```bash
python app.py
